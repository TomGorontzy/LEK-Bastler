# Anwenderprüfungs-Kurzcheckliste

## Ziel

Diese Kurzcheckliste ist für einen kompakten 5–10-Minuten-Test der Windows-EXE gedacht.

Sie eignet sich besonders für Kolleginnen und Kollegen, die schnell beurteilen sollen, ob die Anwendung im Alltag verständlich, stabil und fachlich plausibel wirkt.

Für eine ausführlichere Prüfung siehe `docs/ANWENDERPRUEFUNG_CHECKLISTE.md`.

## Vorbereitung

Bitte vor dem Test sicherstellen:

- Das Windows-Release ist vollständig entpackt.
- Die EXE lässt sich lokal starten.
- Mindestens eine realistische Aufgabensammlung ist verfügbar.

## Kurztest in 6 Schritten

### 1. Start

Bitte prüfen:

- Lässt sich die EXE ohne Probleme starten?
- Ist direkt verständlich, was als Nächstes zu tun ist?

Bewertung:

- [ ] unauffällig
- [ ] teilweise unklar
- [ ] problematisch

## 2. Datei laden

Bitte prüfen:

- Ist das Laden einer Datei leicht auffindbar?
- Reagiert die Anwendung nach dem Laden nachvollziehbar?

Bewertung:

- [ ] unauffällig
- [ ] teilweise unklar
- [ ] problematisch

## 3. Aufgaben prüfen

Bitte prüfen:

- Wirken die erkannten Aufgaben plausibel?
- Werden Intro-/Hinweistexte nicht fälschlich als Aufgabe behandelt?

Bewertung:

- [ ] unauffällig
- [ ] teilweise unklar
- [ ] problematisch

## 4. Vorschau und Navigation

Bitte prüfen:

- Ist klar, wie man weitergeht?
- Ist die Vorschau verständlich und fachlich plausibel?

Bewertung:

- [ ] unauffällig
- [ ] teilweise unklar
- [ ] problematisch

## 5. Export

Bitte prüfen:

- Lässt sich der Export leicht auslösen?
- Entspricht die erzeugte LEK grob der erwarteten Vorschau?

Bewertung:

- [ ] unauffällig
- [ ] teilweise unklar
- [ ] problematisch

## 6. Gesamteindruck

Bitte prüfen:

- Würden Sie die Anwendung ohne weitere Erklärung erneut nutzen?
- Würden Sie sie einer Kollegin oder einem Kollegen empfehlen?

Gesamturteil:

- [ ] sofort alltagstauglich
- [ ] grundsätzlich brauchbar
- [ ] nur mit Erklärung gut nutzbar
- [ ] aktuell noch zu problematisch
